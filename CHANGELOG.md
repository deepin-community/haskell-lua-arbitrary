# Changelog

`lua-arbitrary` uses [PVP Versioning][].

## lua-arbitrary-1.0.1

Released 2022-02-19.

-   Allow lua-2.2.0 (includes Lua 5.4).

-   Implemented `shrink` for Lua integers and numbers.

-   Explicitly disable shrinking for TypeCode, StatusCode, and
    OPCode.

## lua-arbitrary-1.0.0

Released 29-01-2022.

Extracted from hslua-1.3.0.
